# Kho tài liệu nội bộ Techla AI

Thêm file `.md` vào thư mục này để được index bởi tool `tim_trong_tai_lieu`.

## Quy ước

- Tên file: kebab-case, không dấu (vd: `chinh-sach-nghi-phep.md`)
- Mỗi file nên có tiêu đề `#` ở đầu
- Chia đoạn bằng dòng trống để search tốt hơn
- Dùng keywords rõ ràng trong nội dung để LLM dễ tìm

## Ví dụ file có sẵn

- `chinh-sach-nghi-phep.md` — quy định nghỉ phép
- `quy-trinh-onboarding.md` — onboarding nhân viên mới
